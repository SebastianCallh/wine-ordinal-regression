## Bayesian ordinal regression
This repo contains a Bayesian ordinal regression model that predicts wine quality.
A description of the data be found [here](https://www.kaggle.com/datasets/ruthgn/wine-quality-data-set-red-white-wine).

## Running the code
`pip install -r requirements.txt` into a clean python environment, and then simply run `python scripts/main.py` to produce plots and inference data.
